## Prerequite

python -m venv .venv<br>
source .venv/bin/activate<br>
sudo apt-get install portaudio19-dev<br>
sudo apt-get install ffmpeg<br>
pip install -r requirements.txt<br>

## Steps to build

    (프로젝트 실행을 위한 빌드 절차 기술)

cd ~/xxxx
source .venv/bin/activate

make
make install

## Steps to run
